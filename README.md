# test-design

범정부 UI/UX 가이드를 준수하는 디자인 시스템 라이브러리입니다.

## 설치

```bash
npm install
```

## 빌드

```bash
npm run build
```

## 서브모듈로 사용

이 프로젝트는 서브모듈로 사용할 수 있도록 설계되었습니다.

```bash
git submodule add https://github.com/jyh2610/test-design.git
```

## 주요 구성 요소

- Button
- Input
- Text
- ColorBox

## 디자인 토큰

`tokens.json` 파일에 디자인 토큰이 정의되어 있습니다.
